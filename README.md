# hiwaygo

A mobile application for highway bus booking and real-time tracking. Passengers can search routes, book seats, make payments, 
and track buses live, while drivers can manage schedules and share location updates. Built using Flutter with a .NET backend 
and PostgreSQL database to ensure convenience, reliability, and transparency for intercity travel.
